Paragraph Design for Web (Cont.)

  • Take a look at the web pages your classmates designed and coded for homework last week. You can browse them at the bottom of this page.
  • Choose THREE of your classmates’ web pages to download. This week, you will edit each web page’s code. Your goal is to spend the whole week dramatically changing and improving the design of each one.
  • Improve the design: This is part subjective; improve it guided by your personal taste. It is also part objective; after having a week to practice with your own page, you are now devoting a whole extra week to improve the design of existing ones. I expect each new web page to look different, and to look roughly 5 times more beautiful than the original.
  • Change the design: Style the page so that it dramatically contrasts with the original. For example, if the original had small font-sizes, use large font-sizes. If the original used sans-serifs, use serifs. It the original’s text alignment was centered, left-align it. If the original had tight line spacing, increase the line spacing...
  • Edit the code: Step one of editing your classmate’s code will be to “clean it up”. That means indenting and spacing the code so it appears more organized and is easier to comprehend. It also means making sure only the content that should be in the <head></head> or <body></body> sections are in those sections. It also means making sure all “opening tags” like a <div>, <p>, or <h1> are eventually followed by closing tags like a </div>, </p>, or </h1>.
  • For inspiration: Flip through the case studies in Designing the Editorial Experience. See a design you like? Copy it.
  • For coding resources: Browse all of different kinds of styling techniques at W3 Schools. I want you to explore new fonts, line-spacing, top and bottom margins. Email me if you cannot find what you need.
  • Don't forget: Lay out and design the quote for mobile. (I emailed you last week with instructions on how to view your page for mobile on Chrome).
  • Your HTML and CSS web files should be saved in a project folder. Label the project folder as your first name (ex: dylan – one word, all lowercase). Label the HTML files exactly like so: index-1.html, index-2.html, index-3.html. Label each one’s corresponding CSS page exactly like so: styles-1.css, styles-2.css, styles-3.css (and make sure that in the <head> section of each HTML page, it correctly links to its corresponding CSS page).
  • Upload the folder to our shared google drive folder by 10:00AM next Thursday.
