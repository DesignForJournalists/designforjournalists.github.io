Workshop: “Coming Soon” publication Sites

  • Take a quick first glance at each others' pages. No reading - just a split-second up-and-down. Write down 5 words (values or emotions) that come to mind when seeing this page.
  • Now, read the page. Discuss: how does the content lineup with the look and feel? Do you have any suggestions?
  • Think about the user experience once they arrive. Now that the user has landed here and read the letter, what do you want them to do? Should they just leave, or should they do something here? If so, add it.
  • Think about the user experience before they arrive. How do they land on this page? From a tweet? From a newsletter? From an article someone else is writing about the page? Are there any image graphics or lines of copy that need to be created for this, like a Twitter default share text and an image, a simple sentence that summarizes your publication idea's value, or a graphic someone can include in an article? Make it.

The page must also do the following things. If you get through the above, get to these next:

  • be optimized for mobile
  • include information at the bottom detailing: that this is a class assignment, the name of the class, the year/semester, the instructor.
