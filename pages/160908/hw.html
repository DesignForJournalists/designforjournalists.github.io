09-08 Assignment

Paragraph Design for Web

  • Collect 3 inspiring quotes, varying from 500 – 1000 characters long. For good quotes, check out Maria Papova’s Brain Pickings.
  • Lay out all the quotes in one web page, and design each quote in a different way.
  • The page should have a header, a subheader that explains the assignment, and your name as the designer/editor of the page.
  • Each quote should have a header, the quote itself, and an attribute.
  • Think about: balance, legibility, dynamism, softness, loudness, and contrast... Play with: font size, scale, leading, alignments, contrast, margins...
  • Your HTML and CSS web files should be saved in a project folder. Label the project folder as your first name (ex: dylan – one word, all lowercase). Label the html file exactly like so: index.html. (Again, one word, all lowercase.)
  • Upload the folder to our shared google drive folder by 10:00AM next Thursday.
