Feb 02 Assignment

Paragraph Design for Web

  • Collect 1 inspiring quote, varying from 500 – 1000 characters long. For good quotes, check out Maria Papova’s Brain Pickings.
  • Lay out and design the quote as web page.
  • Each quote should have: a header, a lede that introduces the quote, the quote itself, and an attribute. At the bottom of the page, include a section with: The name of the assignment, the class name, the instructor name, your name, and the source where you found the quote.
  • Clearly and elegently apply different styles to different elements if you have good reason to visually distinguish them.
  • Be good to yourself and others. Annotate your document with "commented-out" text to explain how different parts of your code functions.
  • Your HTML and CSS web files should be saved in a project folder. Label the project folder as your first name (ex: dylan – one word, all lowercase). Label the html file exactly like so: index.html.
  • Upload the folder to our shared google drive folder by 10:00AM next Thursday.
