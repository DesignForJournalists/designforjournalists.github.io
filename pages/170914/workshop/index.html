Margin vs Padding

Example

Say you ordered three end tables from Ikea for delivery. Right now, the end tables are in a delivery truck on their way to your door. Each end tables is packaged in its own perfect-sized box. The boxes are side-by-side in the truck, with a 2-inch space between the outside of each box. The inside of each box is fully lined with a layer of bubble wrap that is 1-inch thick; there is a 1-inch space between the inside of each box and the end-table.

We can say that each box has both a margin and a padding. The box margin is the outer space – between the box wall and another box. The padding is the inner space – between the box wall and the end table.

In Code

Think of the box as a tag element – like a div (or a p or an h1). In your css, you can give the div a margin value and a css value.
