Design Assignment

Your 2/11 assignment was to design five inspiring quotes on five 8 x 10 inch pages. Do the same thing, but now, as five web pages.

  • You can use the same quotes you used before, or different quotes. All the designs should be new.
  • This time, you can use color, and you can use multiple fonts per composition. For colors, google "css color hex picker" and for fonts, google "web safe fonts". W3Schools is the online resource for both.
  • Submit 5 different folders, each with their own html and css file. Label each folder as firstname-1, firstname-2, firstname-3, etc. Label the html file in each folder index.html
  • Upload all five folders to our shared google drive Assignments folder by 11:00AM next Thursday.

Reading Assignment

Read the following chapters of Designing the Editorial Experience by Sue Apfelbaum and Juliette Cessar:

  • Introduction
  • Anatomy
  • Typography
  • Layout
